This repository contains the output logs of Claude, Copilot, and Codex when run on a set of github course project repositories to complete the assignments.  
This is for the class CS598 at UIUC in the Spring of 2026.  
We used these logs to analyze the behaviors of these coding agents.  Our results are in our final paper submission: My Coding Agent Did My Homework: Evaluating Code Quality and Behavior of Coding Agents on GitHub CS Projects
