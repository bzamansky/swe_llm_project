Project Summary

This repo is a small Hoare-logic teaching project: it models a tiny imperative language, logical assertions, weakest preconditions, and verification conditions for proving simple programs correct. The assignment brief says students are expected to implement wprec in WPrec.py (line 14) and VC in VCs.py (line 13), while VCG is a more optimized VC generator scaffolded in the same file. The overall intent is stated in README.md (line 4).

The codebase is organized around:

Arithmetic and boolean ASTs in Exprs.py (line 6)
Imperative commands like Skip, Assgn, Seq, IfThen, While in Imp.py (line 6)
Logical/specification formulas in Specs.py (line 7)
Substitution and boolean-to-spec conversion in Specs.py (line 356) and Specs.py (line 451)
Optional Z3 proof checking in Z3Driver.py (line 1)
One important note: this repository is currently a scaffold, not a finished tool. wprec and VC still contain pass placeholders in WPrec.py (line 18) and VCs.py (line 18).

How To Run It

The README requires Python 3.10 and colorama, with Z3 optional for theorem proving in README.md (line 9). The intended commands are:

python3.10 test_WPrec.py
python3.10 test_VCs.py
python3.10 test_Z3Driver.py
