This repository is a starter scaffold for HW1 of CS598: build a minimal command-line coding agent called nanocli that talks to an OpenAI-compatible Responses API, detects exactly one fenced bash tool call in the model output, optionally asks for approval, runs that shell command locally, feeds the command output back into the conversation, and repeats until the model returns plain assistant text. The assignment spec is in hw1.md, the thin terminal UI is in src/nanocli/cli.py, and the unfinished core loop is in src/nanocli/core.py. The tests in tests/test_core.py are effectively the behavioral spec.

How it should be run:

Install dependencies with uv; this project is configured in pyproject.toml.
Run tests with uv run pytest.
Run the mock API server with uv run nanocli-mock.
In another terminal, point the client at the mock server:
export OPENAI_BASE_URL=http://localhost:8000/v1
export OPENAI_API_KEY=dummy
uv run nanocli --model mock --yolo
For interactive use against a real compatible backend, run uv run nanocli --model <model-name> and optionally --instructions-as-system or --reasoning_effort ....
