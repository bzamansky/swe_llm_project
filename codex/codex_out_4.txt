What It Is

This repository is a course assignments repo for CS 769 / Advanced NLP, not one single end-user app. The top level contains four separate deliverables plus a project rubric:

assignment1: a PyTorch deep averaging network sentiment classifier over SST and CFIMDB datasets, with student TODOs in the model/training pipeline. See assignment1/README.md and assignment1/main.py.
assignment2: a “mini-BERT” assignment where students implement key BERT components, an AdamW optimizer, and a sentence classifier for SST/CFIMDB. See assignment2/README.md and assignment2/structure.md.
assignment3: a project proposal submission spec only; no runnable code scaffold beyond the README. See assignment3/README.md.
assignment4: a toy beam search implementation and sanity tests. See assignment4/README.md and assignment4/beam_search.py.
project: course project report/presentation requirements only, not executable code. See project/README.md.
One important caveat: several files still contain NotImplementedError, so this repo is intended as student starter code rather than a fully runnable finished solution.

How To Run It

Run each assignment from its own directory.

Assignment 1:

Main command: python main.py --train data/sst-train.txt --dev data/sst-dev.txt --test data/sst-test.txt ...
Batch training entrypoint: bash run_exp.sh
Optional setup in the script downloads GloVe with wget and unzip.
Key files: assignment1/run_exp.sh
Assignment 2:

Environment setup: bash setup.sh
Training/testing entrypoint: python classifier.py --option pretrain|finetune --epochs ... --lr ... --train ... --dev ... --test ...
Batch experiment script: bash run_exp.sh
Sanity tests: python sanity_check.py and python optimizer_test.py
Key files: assignment2/setup.sh, assignment2/run_exp.sh
Assignment 3:

No code to run; it is a submission/report spec.
Assignment 4:

Test command: python test_beam_search.py
Key file: assignment4/test_beam_search.py
Project folder:

No executable workflow is defined; it documents report/presentation expectations only.
