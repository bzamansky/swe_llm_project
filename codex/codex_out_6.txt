Project
This repository is a course starter project for a Python command-line app that explores NASA near-Earth object data. The intended finished tool loads a CSV of NEO metadata and a JSON file of close-approach records, builds Python objects, links NEOs to their approaches, lets you inspect a single object, query approaches with filters, and optionally export results as CSV or JSON. The main entry point is main.py, and the assignment/spec is documented in README.md.

One important caveat: this repo is intentionally incomplete. Core implementation files like models.py, extract.py, database.py, filters.py, and write.py still contain TODOs, so the intended CLI exists, but the full functionality is not implemented yet.
