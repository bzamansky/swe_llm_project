What It Is

This repository is a hands-on Python workshop on vectorization with NumPy and Pandas. It gives seven small exercises (q1.py through q7.py) where participants replace slow or placeholder logic with vectorized implementations, then compare correctness and speed against baseline versions using shared timing helpers in util.py and the runner in test_run.py. The README explicitly frames it as a “vectorization workshop” with a local-Python or Google Colab workflow in README.md.

The exercises progress from simple data structure conversion to array math and conditional logic:

q1.py: list to np.ndarray
q2.py: dict to pd.DataFrame
q3.py: power/exponentiation
q4.py: vector addition
q5.py: numeric grades to letter grades
q6.py: pass/fail mapping
q7.py: row-wise insurance factor logic over a DataFrame
At the moment, all vec_*/conversion functions are placeholders with pass, and python3 test_run.py confirms nothing is implemented yet.

How To Run It

Local setup from the README:

python3.9 -m venv venv
source venv/bin/activate
python -m pip install -r requirements.txt
Implement the missing functions in the q#.py files.
Run python test_run.py
Colab setup from the README:

Open a new notebook.
Paste the helper code from util.py.
Paste an exercise file’s code, excluding from util import ...
Fill in the vec_* function and run that exercise’s test_* function manually.
