What It Is

This repo is primarily a course repository for SE 444, and the only runnable software project in it is RoboMind, a Python teaching project where students build an AI agent for a 2D grid world. The intended progression is:

Implement search algorithms: BFS, UCS, A*
Add logic-based reasoning with a knowledge base
Add probabilistic reasoning with Bayes’ rule
Combine them into a hybrid rational agent
The intended structure is described in README.md and PROJECT_OVERVIEW.md. In actual code, the grid-world environment and CLI are implemented, but most of the AI pieces are still starter skeletons: main.py (line 56), environment.py (line 39), search_algorithms.py (line 18), knowledge_base.py (line 12), bayes_reasoning.py (line 12).
