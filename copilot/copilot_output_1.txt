**Purpose**

This repository is a starter homework project for building a minimal command-line code agent called nanocli. The intended behavior is:

1. Accept a user prompt in a terminal UI.
2. Send the conversation to an LLM through the OpenAI Responses API.
3. Detect whether the model replied with a fenced bash block.
4. If so, optionally ask for approval, run the shell command, capture stdout/stderr/exit code, and feed that result back into the conversation.
5. Repeat until the model returns normal assistant text with no new bash block.

That design is described in hw1.md, and the actual scaffold lives in core.py and cli.py.

One important detail: this repo is intentionally incomplete. The three core functions in core.py still raise NotImplementedError, so the project is meant to be finished by implementing:
- parse_bash_block
- run_bash_command
- run_turn
