**Purpose**

This repository is a course repository for SE 444: Introduction to Artificial Intelligence, not a single production application. The top level is mostly teaching material: lecture slides, assessments, syllabus/admin documents, and supporting resources, as described in README.md, README.md, README.md, and README.md.

The main runnable project is RoboMind in README.md. Its intended purpose is to let students build a rational agent in a 2D grid world that combines:
1. Search algorithms such as BFS, UCS, and A*.
2. Logic-based reasoning with a knowledge base.
3. Probabilistic reasoning with Bayes’ rule.
4. A hybrid agent that integrates all three.

There are also lecture lab exercises under Lecture 03 for search algorithms. Those are smaller teaching artifacts: a grid pathfinding lab and an 8-puzzle lab, mainly meant for Jupyter or Colab use via README.md and README.md.

**How It Should Be Run**

For the repository as a whole, there is no single global start command. Most folders are static content. The runnable parts are:

1. RoboMind
   From QUICKSTART.md and main.py:
   - cd projects/robomind_project
   - pip install -r requirements.txt
   - python main.py --demo
   - python main.py --test-search
   - python main.py --test-logic
   - python main.py --test-probability
   - python main.py --test-hybrid
   - python main.py --experiment all

   In the current state of the repo, the intended CLI is present, but several modes are still scaffolding. The environment demo is the most clearly runnable path. The search, logic, probability, and hybrid paths are structured for student implementation and some modules still raise NotImplementedError in search_algorithms.py, logic_agent.py, probabilistic_agent.py, and hybrid_agent.py.

2. Lecture 03 labs
   The intended usage is notebook-first:
   - Open grid_search_lab.ipynb in Jupyter or Google Colab
   - Open puzzle_search_lab.ipynb in Jupyter or Google Colab

   There are also script/demo paths documented in README_INTERACTIVE.md:
