**What It Is**

This repository is an educational AI project called RoboMind. Its intended goal is to let students build a rational agent that navigates a 2D grid world by combining three AI techniques:

1. Search algorithms: BFS, UCS, A*
2. Logic-based reasoning: a propositional knowledge base with forward chaining
3. Probabilistic reasoning: Bayesian belief updates for noisy sensors

That intent is stated in README.md and PROJECT_OVERVIEW.md.

The project is best understood as a starter scaffold, not a finished system:
- The simulator and CLI runner are implemented in environment.py and main.py.
- Most AI modules are still student TODOs in search_algorithms.py, knowledge_base.py, bayes_reasoning.py, and the agent files under agents.

**How It’s Supposed To Work**

The intended workflow is:

1. Run the grid-world demo to understand the environment.
2. Implement search in search_algorithms.py.
3. Build logic inference in knowledge_base.py.
4. Build Bayesian updates in bayes_reasoning.py.
5. Integrate everything in hybrid_agent.py.

The CLI wiring for those phases is in main.py.

**How To Run It**

From the project root:

1. Check Python:
   `python --version`

2. Install dependencies:
   `pip install -r requirements.txt`

3. Run the interactive environment demo:
   `python main.py --demo`

4. Run the phase test entry points:
   `python main.py --test-search`
   `python main.py --test-logic`
   `python main.py --test-probability`
   `python main.py --test-hybrid`

5. Run module-level test scripts mentioned in the docs:
   `python ai_core/search_algorithms.py`
   `python ai_core/knowledge_base.py`
   `python ai_core/bayes_reasoning.py`

These commands are documented in QUICKSTART.md, README.md, and PROJECT_OVERVIEW.md.
