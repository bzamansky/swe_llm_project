This repository is an educational Hoare-logic verification project, not a finished application. Its purpose is to let students implement weakest-precondition generation and verification-condition generation for a small imperative language, then optionally check those proof obligations with Z3. That intent is stated in README.md, and the missing student work is explicit in WPrec.py and VCs.py.

**What the project is intended to do**
- Model arithmetic and boolean expressions as AST nodes in Exprs.py.
- Model logical assertions and substitution/conversion helpers in Specs.py.
- Model a toy imperative language with `Skip`, assignment, sequencing, conditionals, and loops in Imp.py.
- Compute:
  1. `wprec(c, q)`: the weakest precondition of command `c` with respect to postcondition `q` in WPrec.py
  2. `VC(pre, c, post)`: verification conditions for a Hoare triple in VCs.py
  3. `VCG(pre, c, post)`: a more optimized VC generator in VCs.py
- Optionally translate those logical formulas to Z3 and ask the solver to prove them in Z3Driver.py.

In practice, the project is about proving statements of the form $\{P\}\ C\ \{Q\}$ for simple programs.

The repo is set up to be run as plain Python scripts, mainly through the test files listed in README.md.
