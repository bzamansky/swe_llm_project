**Overview**
This repository is a CS61C-style RISC-V assembly assignment for a small matrix-based classifier. The intended pipeline is:

1. Read two pretrained weight matrices and one input matrix from binary files.
2. Compute a first linear layer: `m0 * input`.
3. Apply `ReLU` in place.
4. Compute a second linear layer: `m1 * hidden`.
5. Write the final output matrix to disk.
6. Run `argmax` on that output to produce the predicted label, and print it when requested.

The core program flow is defined in main.s and classify.s. The matrix file format and I/O responsibilities are documented in read_matrix.s and write_matrix.s.

**What Each Source File Is For**
- abs.s: absolute value of one integer.
- relu.s: in-place ReLU on an integer array.
- argmax.s: index of the largest element.
- dot.s: dot product with configurable strides.
- matmul.s: matrix multiplication built on `dot`.
- read_matrix.s: open/read/allocate matrix from binary file.
- write_matrix.s: serialize matrix to binary file.
- classify.s: orchestrates the full inference pipeline.
- utils.s: provided runtime helpers and Venus syscall wrappers.
