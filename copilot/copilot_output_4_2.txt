**Purpose**

This repository is a course assignment for building a minimal BERT implementation in PyTorch and using it for sentence classification on SST and CFIMDB.

The intended workflow is:

1. Implement the missing BERT pieces in bert.py, the classifier head in classifier.py, and AdamW in optimizer.py.
2. Verify the implementation with sanity_check.py and optimizer_test.py.
3. Train/evaluate on SST and CFIMDB through classifier.py or run_exp.sh.
4. Package outputs with prepare_submit.py.

The project is not a finished training package as checked in. It is an instructional skeleton: several core methods still raise `NotImplementedError` in bert.py, classifier.py, and optimizer.py.

**How It Should Be Run**

Environment setup is documented in setup.sh. It expects a Conda environment and installs pinned dependencies:

```bash
bash setup.sh
```

The canonical training command from README.md is:

```bash
python3 classifier.py \
  --option [pretrain/finetune] \
  --epochs NUM_EPOCHS \
  --lr LR \
  --train data/sst-train.txt \
  --dev data/sst-dev.txt \
  --test data/sst-test.txt
```

The repo’s own experiment script in run_exp.sh runs fine-tuning on both datasets and writes logs, predictions, and checkpoints:

```bash
bash run_exp.sh
```

More explicitly, it calls commands of this form:

```bash
python classifier.py \
  --use_gpu \
  --option finetune \
  --lr 1e-5 \
  --seed 1234 \
  --train data/sst-train.txt \
  --dev data/sst-dev.txt \
  --test data/sst-test.txt \
  --dev_out CAMPUSID/sst-dev-output.txt \
  --test_out CAMPUSID/sst-test-output.txt \
  --filepath CAMPUSID/sst-model.pt
```

and similarly for `cfimdb`.

Validation commands:

```bash
python sanity_check.py
python optimizer_test.py
```

Submission packaging:

```bash
python prepare_submit.py path/to/output_dir CAMPUSID
```

or to validate an existing zip:

```bash
python prepare_submit.py path/to/submission.zip CAMPUSID
```

Behavior:

- `finetune` means BERT weights are updated.
- `train(args)` is always followed by `test(args)` in the script entrypoint.
