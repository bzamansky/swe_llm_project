This repository is an assignment scaffold for building and evaluating image classifiers on a reduced CIFAR-10 task. The intended end goal is in assignment.py: load CIFAR data for two classes, train either a CNN or MLP, and reach at least 70% test accuracy on the cat-vs-dog subset. A second, lower-level part of the assignment is in manual_convolution.py, where you implement a custom convolution layer and validate it against Keras in local_test.py.

The code is not complete yet, so this is more “how it is supposed to be run” than “what runs successfully right now.” The intended workflow is:
1. Put the CIFAR pickled data where assignment.py expects it for autograding, or fill in the local paths in assignment.py.
2. Implement preprocessing in preprocess.py, shared loss/accuracy in base_model.py, model forward passes in cnn.py and/or mlp.py, and the training/testing loops in assignment.py.
3. Run `python local_test.py` to check the custom convolution layer against `keras.layers.Conv2D`.
4. Run `python assignment.py` to train and evaluate the classifier.

The main internal APIs are straightforward. preprocess.py defines `unpickle`, `get_next_batch`, and `get_data` for loading CIFAR batches, filtering selected classes, and producing one-hot labels. base_model.py defines an abstract `CifarModel` with `call`, `loss`, and `accuracy`. cnn.py and mlp.py are intended concrete subclasses. assignment.py defines the training loop, testing loop, visualization helpers, and top-level `main()`. manual_convolution.py defines `ManualConv2d`, which is then exercised by local_test.py.

The libraries actually used or referenced are:
- `tensorflow` and `tf.keras` for models, layers, variables, tensors, one-hot labels, image augmentation, and optimization
- `keras.layers.Conv2D` in local_test.py as the reference implementation
- `numpy` for array handling and test comparisons
- `matplotlib.pyplot` for loss/result visualization in assignment.py
- `pickle` for reading CIFAR batch files in preprocess.py
- Python standard library modules `os`, `math`, `random`, and `abc`
