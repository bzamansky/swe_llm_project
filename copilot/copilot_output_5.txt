**Purpose**

This repository is a partially completed course assignment for image classification on a 2-class subset of CIFAR-10, specifically cat vs. dog.

The intended work is split across these modules:

- assignment.py defines the top-level training loop, test loop, visualizations, and the main entry point.
- base_model.py defines a common TensorFlow model base class with abstract `call`, plus `loss` and `accuracy` helpers students are expected to implement.
- mlp.py is a scaffold for a multilayer perceptron baseline.
- cnn.py is a scaffold for a convolutional neural network.
- manual_convolution.py asks the student to implement a custom 2D convolution layer manually.
- preprocess.py is meant to load CIFAR pickled batches, filter to chosen classes, normalize images, and one-hot encode labels.
- local_test.py is a local sanity-test harness that compares the custom convolution layer against Keras `Conv2D`.

In short: the project is intended to teach how to build and train an MLP/CNN in TensorFlow and how convolution works internally by implementing it by hand.
