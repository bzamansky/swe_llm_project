**Project Purpose**
This repository is a starter template for a Python autograded assignment in GitHub Classroom, not a finished application.

From README.md, the intended workflow is:

- Teachers use the repo as a template for a private classroom assignment.
- Students implement solutions in assignment.py.
- Tests live in test_assignment.py and are meant to be inspected but not changed by students.
- GitHub Classroom runs the tests automatically through GitHub Actions and the classroom autograder config.

In its current form, the assignment asks students to implement three functions in assignment.py:

- `hello_world() -> str`
- `add_two(number: int) -> int`
- `zero_array(N: int) -> np.ndarray`
