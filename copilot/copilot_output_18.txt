This repository is a template for a Python assignment meant to be distributed and autograded through GitHub Classroom. The intent is described in README.md: students implement functions in assignment.py, and grading is driven by tests in test_assignment.py. In its current state, the assignment asks students to implement three functions: return Hello World, add 2 to an integer, and return a NumPy zero-filled array of length N.

To run it locally, install the listed dependencies from requirements.txt, then run pytest against the test file. The repository’s own autograder config in autograding.json uses:
1. sudo -H pip3 install -r requirements.txt
2. pytest test_assignment.py::test_hello
3. pytest test_assignment.py::test_add_two
4. pytest test_assignment.py::test_zero_array

The GitHub-side automation is wired through classroom.yaml, which triggers on pushes that modify assignment.py or test_assignment.py, then runs the GitHub Classroom autograding action.
