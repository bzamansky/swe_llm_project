**Purpose**

This repo is a course project for combining RSA cryptography with DWT steganography to secure text messages inside digital images. The stated scope in README.md is RSA for encryption, DWT for steganography, text payloads, and image covers; the evaluation focus is RSA timing and avalanche effect plus DWT payload capacity and robustness in README.md. The main end-to-end demo in main.py does exactly that: generate RSA keys, encrypt a text message, embed the ciphertext into an image with DWT, extract it back out, and decrypt it.

There are also extra audio experiments beyond the original README scope. main_image.py is the clean image-only demo. main_dwt.py, main_lsb.py, and main_echo-hiding.py are audio-cover variants using DWT, LSB, echo hiding, and DCT-based embedding. The notebooks in notebooks are for benchmarking and analysis rather than the core runtime path.

**How To Run**

The intended basic flow is the one described in README.md:

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python main.py
```

Useful entry points:

1. `python main.py`
   Runs the primary RSA + DWT image steganography demo.
2. `python main_image.py`
   Runs the simpler image-only variant explicitly.
3. `python main_dwt.py`
   Runs the audio DWT variant.
4. `python main_lsb.py`
   Runs the audio LSB variant.
5. `python main_echo-hiding.py`
   Runs the audio echo-hiding variant.

Sample assets already exist under cover, including `cover1.jpg` and `cover1.wav`, and outputs are written under stego. For analysis work, the notebook workflow is centered on 01_rsa_benchmark.ipynb, 02_dwt_payload_psnr.ipynb, 03_robustness_tests.ipynb, and 04_avalanche_effect.ipynb.
