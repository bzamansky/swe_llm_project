**Purpose**

This repository is a CS 769 assignment scaffold for building a neural text classifier, specifically a Deep Averaging Network for sentiment classification. The assignment description and expected deliverables are in README.md. The training pipeline is in main.py, the model skeleton is in model.py, and vocabulary construction is in vocab.py.

What it is intended to do:

- Train a sentence-level sentiment classifier on two datasets under data:
  - SST
  - CF-IMDb
- Read examples formatted as `label ||| tokenized text`, lowercase them, convert tokens to ids, train a model, and write predicted labels for dev/test sets. The dataset reader is in main.py.
- Save the best model checkpoint during training and then generate output files for submission. Evaluation/output writing is in main.py, and checkpoint save/load is in model.py.

- This repo is a partially completed starter project, not a finished application.

**How To Run**

The intended workflow is documented in README.md and automated in run_exp.sh.

Primary commands:

- Run one experiment directly:
```bash
python main.py \
  --train data/sst-train.txt \
  --dev data/sst-dev.txt \
  --test data/sst-test.txt \
  --dev_output output.dev.txt \
  --test_output output.test.txt
```

- Run the provided experiment script for both datasets:
```bash
bash run_exp.sh
```

What run_exp.sh does:

- Sets a `CAMPUSID` output directory.
- Optionally downloads GloVe with `wget` and extracts it with `unzip` at run_exp.sh.
- Runs `python main.py` once for SST at run_exp.sh and once for CF-IMDb at run_exp.sh.
- Copies required source files into the submission directory at run_exp.sh.
- Builds/checks the submission zip with `python prepare_submit.py ...` at run_exp.sh.
