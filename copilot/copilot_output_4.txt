**What This Repo Is**

This repository is a bundle of course assignments for CS 769: Advanced NLP, not a single application. The top-level README is effectively empty, and the real intent lives in the assignment folders.

- Assignment 1 is a sentiment-classification exercise built around a Deep Averaging Network on SST and CFIMDB. Students are expected to complete missing PyTorch code in model.py and main.py, following the spec in README.md.
- Assignment 2 is a minimalist BERT implementation plus sentence classification on the same SST and CFIMDB datasets. Students implement pieces of self-attention, transformer layers, embeddings, and AdamW, using the structure described in README.md and structure.md.
- Assignment 3 is not code; it is a project proposal brief in README.md.
- Assignment 4 is a beam-search programming exercise for a toy encoder-decoder model in beam_search.py, validated by test_beam_search.py.
- The project folder is final course-project/report guidance, again mostly documentation, in README.md.

The overall purpose is educational: each folder is a partially completed NLP assignment with datasets, starter code, and submission helpers.

**How To Run It**

There is no single repo-wide entrypoint. You run each assignment from its own directory.

- Assignment 1:
  Run the training script directly with python main.py and dataset/output arguments, or use the canned workflow in run_exp.sh. That script also optionally downloads GloVe with wget and unzip, then packages a submission with python prepare_submit.py.
- Assignment 2:
  First create the environment from setup.sh. Then run python classifier.py with options like pretrain or finetune, train/dev/test paths, output files, and a checkpoint path. The provided batch script is run_exp.sh. There are also validation commands:
  python sanity_check.py
  python optimizer_test.py
  and packaging via python prepare_submit.py.
- Assignment 4:
  Run python test_beam_search.py from the assignment4 folder. That file exercises topK, hidden-state beam selection, sequence extraction, and the full beam search.
- Assignment 3 and project:
  These are writeup/submission instructions, not runnable software.

Operationally, the repo expects standard shell commands such as mkdir, cp, tee, wget, unzip, conda create, conda activate, conda install, and pip install, mainly in run_exp.sh, run_exp.sh, and setup.sh.
