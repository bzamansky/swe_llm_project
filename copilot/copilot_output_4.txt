## Project intent

- The assignment goal is described in hw4.md: build a CLI agent that detects large context windows using API-reported token usage, compacts history with a fixed fallback prompt, writes a checkpoint summary message, and continues the same task loop.
- The runtime architecture is:
1. CLI handles terminal I/O and approval prompts.
2. Core loop sends conversation context to OpenAI Responses API.
3. Model can emit tool calls.
4. Local tools execute and return structured outputs.
5. Loop repeats until no more tool calls.
- The key compaction behavior and prompts are defined in core.py, including SUMMARIZATION_PROMPT and SUMMARY_TEMPLATE.
