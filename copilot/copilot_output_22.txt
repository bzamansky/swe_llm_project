This repository is a workshop exercise set for learning vectorization with NumPy and pandas. The core idea is: each q1.py through q7.py contains either a missing implementation or a slow baseline function, and the participant is expected to replace the placeholder `pass` in a `vec_*` function with a vectorized solution, then measure correctness and speedups.

**What The Project Is Intended To Do**
- Teach vectorized data processing in Python using NumPy and pandas.
- Compare slow Python loop-based implementations against vectorized versions.
- Let participants report their speedup results as part of the workshop flow described in README.md.

The seven exercises are:
1. q1.py: convert a Python list to a NumPy array.
2. q2.py: convert a dictionary to a pandas DataFrame.
3. q3.py: vectorize exponentiation (`slow_power` vs `vec_power`).
4. q4.py: vectorize elementwise addition of two vectors.
5. q5.py: vectorize numeric-to-letter grade mapping.
6. q6.py: vectorize pass/fail conversion.
7. q7.py: vectorize an insurance-factor calculation over a car DataFrame.

Supporting files:
- util.py: shared timing and reporting helpers.
- test_run.py: sequential runner that executes all exercise test functions.
- slides.pdf: workshop slide deck.
