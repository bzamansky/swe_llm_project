**Purpose**

This repository is a small workshop project for learning NumPy/Pandas vectorization. The seven exercise files q1.py, q2.py, q3.py, q4.py, q5.py, q6.py, and q7.py each ask you to replace a slow loop-based implementation with a vectorized one, then validate correctness and compare runtime. The runner test_run.py just calls all seven `test_*` functions in sequence, and the shared timing/reporting helpers live in util.py and util.py.

The exercises progress from simple conversions to real vectorized transformations:
1. list to NumPy array
2. dict to pandas DataFrame
3. elementwise power
4. elementwise vector addition
5. numeric grades to letter grades
6. letter grades to pass/fail
7. row-wise insurance-factor logic over a DataFrame

**How To Run**

The intended local workflow is documented in README.md: create a virtual environment with `python3.9 -m venv venv` as shown in README.md, activate it, install dependencies with `python -m pip install -r requirements.txt` from README.md, then run all exercises with `python test_run.py` from README.md. There is also a Colab path in README.md, where you paste the helper code into a notebook and run each exercise manually instead of using the script.

There are a few repo quirks worth knowing. README.md says to copy `utils.py` into Colab, but the actual file is util.py. requirements.txt only lists pandas even though the code imports NumPy directly throughout the repo, so NumPy is an undeclared direct dependency. Also, q6.py passes integer grades into `slow_pass_fail`, even though that function expects letter grades, so Q6’s test setup is internally inconsistent.

Main libraries:
- `numpy`
